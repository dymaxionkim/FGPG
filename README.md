# How to use
1. Just go to http://juliabox.org
2. Just Log in and Make new Worksheet.
3. Copy & Paste the Source Code.
4. Upload input4fgpg.csv file into the same directory.
5. Ctrl+Enter in the Source Code's Cell.
6. ...
